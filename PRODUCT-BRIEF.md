# Продуктовый бриф: πоток

**Версия:** 1.0 | **Дата:** 5 апреля 2026 | **Статус:** Рабочий

---

## One-liner

**πоток** — визуальная модель бизнеса с AI-слоем для собственника SMB: загружаешь цифры → за 5 минут видишь полную картину и знаешь, где теряешь деньги.

---

## Проблема

85% малого и среднего бизнеса в России управляют компанией вслепую — данные есть, но цельной картины нет.

- **Excel не работает как BI**: медленно, нет визуализации, нет AI-слоя
- **Иностранный софт ушёл**: Tableau, Power BI, Amplitude недоступны или слишком дороги
- **Нанять аналитика = 80–120 тыс. ₽/мес** — непозволительно для SMB
- **Вопрос «Какой канал реально работает?»** остаётся без ответа неделями
- **~300 000 технологически активных SMB в РФ** работают без BI-инструмента

---

## Целевая аудитория

**Основной сегмент:** Владелец или директор малого/среднего бизнеса в России

- Возраст: 28–50 лет
- Бизнес: 3–100 сотрудников, оборот 3–300 млн ₽/год
- Стек: Google Sheets / 1С + amoCRM / Bitrix24 + Яндекс.Метрика
- Боль: тратит 3–6 часов в неделю на ручную сборку отчётов, не видит узкие места
- Не технарь — не пишет SQL, не настраивает дашборды

**Вторичный сегмент:** Product Manager и Growth Lead в стартапе без выделенного аналитика.

---

## Ценностное предложение

> «Не инструмент для аналитика — инструмент для собственника»

πоток за 5 минут превращает разрозненные цифры бизнеса в связную модель с AI-инсайтами. Пользователь видит, где теряет деньги, и получает конкретные рекомендации — без программиста, без настройки, без порога входа.

**Ключевое обещание:** задал вопросы боту → получил модель бизнеса с узкими местами.

---

## Ключевые возможности

| # | Функция | Что даёт пользователю |
|---|---------|----------------------|
| 1 | **Диалоговый онбординг** | Бот ведёт через 4 шага — не нужно разбираться в интерфейсе |
| 2 | **Визуальная карта бизнеса** | Воронка с экономикой по каждому каналу, наглядно и цельно |
| 3 | **Нишевые бенчмарки** | Сравнение с медианой по отрасли: где выше/ниже нормы |
| 4 | **AI-инсайты** | «Вот 3 места, где вы теряете деньги» — с конкретными цифрами |
| 5 | **Личный кабинет** | История данных, обновление модели, сравнение периодов |
| 6 | **Интеграции (Pro)** | Яндекс.Метрика, amoCRM, Тильда — данные подтягиваются автоматически |
| 7 | **Шаринг** | Кнопка «Поделиться картой» → read-only ссылка для партнёров/инвесторов |

## Отличие от дашбордов и BI-инструментов

| Конкурент | Проблема | Наше отличие |
|-----------|----------|--------------|
| Tableau / Power BI | Ушли из РФ, требуют аналитика | Доступны в РФ, zero-setup |
| Яндекс DataLens | Нет AI, нет модели бизнеса, нужен SQL | Диалог + AI-слой + готовая модель |
| Visiology | Enterprise, от 1 млн ₽/год | Freemium, для SMB, за 5 минут |
| Google Sheets | Нет визуализации, нет AI | Визуал + инсайты + бенчмарки |
| Metabase | Требует сервер, нужен аналитик | Облако, самообслуживание |

**Главная ось дифференциации:** πоток строит **модель бизнеса** (воронка → экономика → узкие места), а не дашборд с графиками. Пользователь получает ответ на вопрос «что делать», а не просто данные.

## Бизнес-модель

**Freemium SaaS** с тремя уровнями:

| Тариф | Цена | Что доступно |
|-------|------|-------------|
| **Free** | 0 ₽ | Полный онбординг, snapshot экономики, 3 AI-инсайта, личный кабинет |
| **Pro** | 990 ₽/мес | Обновление данных, live-интеграции, история изменений |
| **Team** | 2 490 ₽/мес | Командный доступ, несколько моделей, KPI/OKR, экспорт для инвесторов |

**Юнит-экономика (подтверждена исследованием рынка):**
- ARPU: ~1 500 ₽/мес
- LTV: ~45 000 ₽ (churn 3%/мес)
- CAC: 3 000–8 000 ₽ (контент + SEO + виральность)
- LTV/CAC: **6–15×** → юнит-экономика работает

**Путь к 1 000 000 ₽/мес:** ~670 платящих клиентов, реалистично за 18–24 месяца при бюджете 50 000 ₽/мес.

**Дистрибуция:** Яндекс.Директ, контент для основателей, партнёры-интеграторы 1С, виральность через шаринг карты бизнеса.
